# Security Policy

## Supported Scope

Security reports are accepted for this repository's source code, scripts, and release artifacts.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Send a private report to the repository maintainer:

- GitHub: https://github.com/brittytino

Include:

- Vulnerability description
- Reproduction steps
- Impact assessment
- Suggested remediation (if available)

## Disclosure

After validation and fix, coordinated disclosure will be performed.
